总结下基础概念

！!

首先，node.js 为 JavaScript 的运行环境，而 npm (Node Package Manager) 作为默认 的包管理器，下载和更新项目所需的第三方依赖 —— 之前我们使用 npm install --save hexo-blog-encrypt 创建了 A Secret 加密页面。 以面对对象编程语言为例，如果说 JavaScript 对应 Python，node.js 可以理解为python interpreter，npm 则对应 pip。

此外，JavaScript 有一种常见的数据格式 JSON (JavaScript Object Notation)，用于储存不同类型字符串。例如，在 Github Issue 中存储 JSON 格式对象来捕捉小组件的计数。类似的，它可以看作 Python 中的字典，两者之间也可相互转换，如下所示:

JSON 转换为字典 

import json
# 下载到一串 JSON 文本
json_text = '{"name": "Alice", "age": 30, "isAdmin": false}'
# 反序列化成 Python dict
data = json.loads(json_text)
print(data["name"])        # Output: Alice
print(type(data))          # Output: Class: "dict"
